# Steps to get this thing working

## Step 1 -> Data Generation

### Create virtual environment

* python3.10 -m venv .env
* pip install -r requirements.txt

### Create assets folder

* Create directory named assets
* Within it, create 3 separete directories -> input, output, csv
* Place input file within input directory

### Execute player_detection_yolo.py

* Execute with python -m player_position_detection/player_detection_yolo.py
